# Task 02 - Exploratory Data Analysis (EDA)

## 📌 Internship

Prodigy InfoTech - Data Science Internship

## 🎯 Objective

Perform data cleaning and exploratory data analysis on a dataset to identify patterns and relationships.

## 📊 Project Description

In this task, I performed Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns based on different factors such as gender, class, age, and fare.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 🔍 Key Learnings

* Data cleaning and handling missing values
* Exploratory Data Analysis techniques
* Data visualization
* Identifying trends and patterns

## 📊 Key Insights

* Female passengers had higher survival rates
* Passengers in higher classes had better survival chances
* Socio-economic factors influenced survival
